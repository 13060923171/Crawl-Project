# 爬取百度文库

## Before use:

```python
执行setup.bat
```

## Requirements

- Python 3 环境
- 将文件夹放置在一个路径文件夹名中**没有空格**的位置

## 使用说明

用户界面：

![1](.\img\1.png)

输入要爬取的百度文库网页的url地址：

![2](.\img\2.png)

如果没有设置环境变量，需要手动输入本地python.exe文件的绝对路径，如果已经设置环境变量，不需要修改该部分：

![3](.\img\3.png)

如果输入了错误的网址或python路径，会弹窗报错：

![4](.\img\5.png)

选择是否爬取文本内容：

![4](.\img\4.png)

![6](.\img\6.png)
